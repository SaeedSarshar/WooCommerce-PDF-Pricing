WooCommerce PDF Pricing is a plugin that calculates product prices based on PDF page count, size, and quantity.

== Description ==
This plugin allows customers to upload a PDF file and calculates the price based on:
- Number of pages in the PDF
- Selected product size (e.g., A4, A5)
- Quantity ordered

== Installation ==
1. Upload the plugin files to the `/wp-content/plugins/wc-pdf-pricing` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Configure pricing settings under the 'PDF Pricing' menu in WordPress admin.
